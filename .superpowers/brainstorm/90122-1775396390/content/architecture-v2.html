Three Zustand Stores
useCallStore
peers: Peer[]
localStream: MediaStream
spotlightPeerId: string
connectionStates: Map
isMuted: boolean
isCamOff: boolean
useUIStore
isChatOpen: boolean
isParticipantsOpen: boolean
activeModal: string | null
toasts: Toast[]
layout: 'spotlight' | 'grid'
useInteractionStore ✦
activeSpeaker: string
raisedHands: string[]
reactions: Reaction[]
activePoll: Poll | null
pollResponses: Map
typingPeers: string[]
How the stores connect
useCallStore → drives SpotlightView, ThumbnailStrip, ControlBar
useUIStore → drives ChatPanel open/close, layout switching, modals
useInteractionStore → drives reaction overlays on VideoTile, raised hand badges, poll UI, typing dots in chat
