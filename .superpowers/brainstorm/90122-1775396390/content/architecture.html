New file structure. Old files untouched until cutover. TypeScript + Zustand throughout.

New Directory Layout

store/
useCallStore.ts ← Zustand: peers, media, layout
useUIStore.ts ← Zustand: panels, modals, toasts
pages/
HomeV2.tsx ← Split Hero lobby
RoomV2.tsx ← thin orchestrator only
call/
PeerManager.tsx ← WebRTC logic, no UI
MediaController.tsx ← mic/cam/stream logic
SpotlightView.tsx ← pinned speaker tile
ThumbnailStrip.tsx ← scrollable participant strip
ControlBar.tsx ← mic, cam, end, more
ChatPanel.tsx ← persistent side panel
ui/
Button.tsx, Avatar.tsx
VideoTile.tsx, Badge.tsx
components/ ← existing, untouched
